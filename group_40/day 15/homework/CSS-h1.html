CSS

CSS არის შიგა ნაწილის გალამაზება, გაფერადება. css დახმარებით საიტს ვხდით უფრო გაფორმებულს დახვეწილს, რაც ძალიან მნიშვნელოვანი მომხმარებლის გულის მოსაგებად.

HTML

HTML არის ვებ გვერდების სტრუქტურირების ენა, რომლის საშუალებით შექმნილი ტექსტური დოკუმენტის განხორციელებაა შესაძლებელი. HTML გამოიყენება ვებ გვერდების შიგნეული სტრუქტურის და შინაარსის გამოსახვაში, რომელიც კონტენტის ფორმატირებას და სტილის ასახავს. HTML კოდი შედგება ტეგებისაგან, რომელიც გვერდის სტრუქტურის საფუძვლებისთვის გამოიყენება.

JS

Javascript არის ფუნქციონალი. JS საშუალებით საიტს ვაძლევთ სხვადასხვა შესაძლებლობებს, მაგალითად ღილაკზე დაჭერის ფუნქცია.ღილაკზე დაჭერისას ახალ page-ზე გადასვლის ფუნქცია.JS გარეშე საიტი იქნება უფუნქციო და მასზე ვერანაირ მოქმედებას ვერ განვახორციელებთ
